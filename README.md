# Important
- There are hidden files in this directory. DO NOT DELETE!
- Need the newer version of `stow` to get `--dotfiles` to work
	- Won't be an issue on arch systems

# Instructions
- Clone `https://github.com/hyliancode/dotfiles.git` to `$HOME/dotfiles`
- Run `$HOME/dotfiles/install`
- Run `bash -x $HOME/dotfiles/install` for debugging
